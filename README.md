## Echoes

A simple Expo app.

### Requirements
- Node.js 18+
- npm 9+

### Setup
```bash
npm install
```

### Run
```bash
npm run start
# or
npx expo start
```

Open in Expo Go (on device) or an emulator/simulator from the Expo CLI.

### Project structure
- `app/`: screens (file-based routing)
- `components/`: UI components
- `data/`, `lib/`, `theme/`, `types/`: supporting modules
